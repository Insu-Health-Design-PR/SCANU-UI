# Design System Notes

## Visual language
- Near-black background
- Dark elevated panels
- Subtle cyan glow
- Minimal amber/red reserved for signal meaning

## Typography
- Strong page title
- Uppercase micro-labels
- Medium-weight panel headers
- Numeric emphasis only for key metrics

## Motion
- Layout changes should feel smooth and premium
- Avoid noisy motion
- Use quick easing and short duration

## Panel rhythm
- Large main panels
- Tight but breathable spacing
- Right rail supports the main content
